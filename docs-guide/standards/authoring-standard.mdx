# Authoring Standard

> Entry point for all authoring standards. The full standard is a published docs page.
> This file exists in `docs-guide/standards/` for governance discoverability.

---

## Quality bar

Every page must meet:

1. **Factual accuracy** — claims verified against primary sources, not assumed
2. **Voice compliance** — matches the audience voice rules in `docs-guide/standards/voice-and-copy.mdx`
3. **Structural compliance** — follows page taxonomy and composition frameworks
4. **UK English** — no exceptions
5. **No self-reference** — pages do not describe themselves
6. **No questions in headings** — state the fact

---

## Publication gate

A page is not done until:

1. Content is factually verified (veracity pass or SME review)
2. Voice rules are applied (content pass)
3. Render verification passes (Mintlify scoped preview)
4. Links resolve (no broken internal or external links)
5. Frontmatter is complete (all required fields populated)
