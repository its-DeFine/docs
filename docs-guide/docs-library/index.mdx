## Pipelines by concern

Each pipeline page documents the full chain: trigger, validators, generators, output, and consumer. Includes Mermaid diagrams and gap analysis.

| Pipeline | Concern | Gate | Frequency | What it covers |
|----------|---------|------|-----------|----------------|
| [Content Quality](pipelines/content-quality) | copy, structure, grammar | P2 (hard gate) | Per PR | Validators, repair scripts, browser tests |
| [Governance Compliance](pipelines/governance-compliance) | governance | P3 + weekly cron | Per PR + Monday 06:00 UTC | Markers, staleness, map generation |
| [Component Health](pipelines/component-health) | components | P3 | Per PR | Registry, naming, CSS, immutability |

| [Copy and Brand](pipelines/copy-brand) | brand, voice | P3 (advisory) | Per PR | EN-GB, banned words, proper nouns, voice rules |
