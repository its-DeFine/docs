# About Audience Design — Review Guide

**Source**: Synthesised from session research + FIRST-PRINCIPLES-SITEMAP + design.md + tab analysis files

---

## How to review

Each part is a separate file. Review them in order. For each one:

- **APPROVE** — it's correct, move on
- **AMEND** — mostly right, note what needs changing
- **REJECT** — fundamentally wrong, needs rethinking

Mark your verdict at the top of each file.

| 7 | `07-path-validation.md` | Each persona's path + gap notes | Do the paths work? Are the gaps real? |
