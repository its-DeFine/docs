# Livepeer Ecosystem Product Cards

---

## Daydream

**Blurb:**
Open-source engine for real-time AI video generation. Compose node-based workflows that combine generative video, audio, and computer vision — runs locally or in the cloud. Native integrations with OSC, MIDI, Syphon, Spout, and NDI connect it directly into professional creative toolchains (Ableton, TouchDesigner, OBS, Resolume). Powered by Livepeer's decentralised GPU compute network.

**Infra:** `open-source` `self-hosted` `sdk`
**Function:** `generative-video` `realtime-inference` `node-based-workflows` `creative-toolchain-integration`
**Domain:** `video` `ai` `realtime-ai`

---

## Livepeer Studio

**Blurb:**
Managed gateway and API platform for Livepeer's decentralised video infrastructure. REST APIs and a dashboard cover live streaming ingest, video-on-demand transcoding, and delivery — with no infrastructure to operate. Usage-based pricing routes jobs to the open orchestrator network, giving applications cost and scale properties unavailable from centralised providers.

**Infra:** `api` `saas` `gateway`
**Function:** `live-ingest` `vod-transcoding` `video-delivery` `asset-management`
**Domain:** `video`

---

## Frameworks

**Blurb:**
Full-stack video streaming platform built by the MistServer team, combining a self-hosted open-source core with optional hosted processing. Deploy the full stack on your own infrastructure, use Frameworks' hosted tier, or run a hybrid. No vendor lock-in: the infrastructure stack travels with you. Built on Livepeer's decentralised protocol and subsidised by the Livepeer treasury.

**Infra:** `open-source` `self-hosted` `saas` `gateway`
**Function:** `live-ingest` `vod-transcoding` `stream-origin` `multi-protocol-output`
**Domain:** `video`

## Embody Avatars

**Blurb:**
Enterprise AI avatar platform for real-time telepresence and on-demand branded content. Integrates into learning platforms, communication tools, and content pipelines to deliver embodied AI presence at scale — covering tutoring, enterprise telepresence, and branded video without live camera requirements.

**Infra:** `api` `saas`
**Function:** `avatar-generation` `realtime-inference` `lip-sync` `interactive-presence`
**Domain:** `ai` `realtime-ai` `agent`

---

## Tag Taxonomy

### Domain tags
| Tag | Meaning |
|---|---|
| `video` | Core workload is video processing or delivery |
| `ai` | Applies AI models as a primary capability |
| `realtime-ai` | AI inference within the latency constraints of live media |
| `agent` | Delivers autonomous or conversational AI behaviour to end users |

### Infra tags
| Tag | Meaning |
|---|---|
| `api` | Exposes REST/WebSocket APIs for programmatic integration |
| `saas` | Hosted, managed service — no infrastructure to run |
| `gateway` | Routes jobs between applications and Livepeer orchestrators |
| `open-source` | Core is open-source; self-hostable |
| `self-hosted` | Designed for full on-premise/own-infra deployment |
| `sdk` | Provides SDKs or embeddable libraries for integration |

### Function tags
| Tag | Meaning |
|---|---|
| `live-ingest` | Accepts RTMP/SRT/WebRTC streams for real-time processing |
| `vod-transcoding` | Processes uploaded video files into delivery-ready formats |
| `video-delivery` | Distributes video to end users via CDN or playback APIs |
| `asset-management` | Stores, organises, and manages video assets and metadata |
| `stream-origin` | Acts as origin server for live stream distribution |
| `multi-protocol-output` | Outputs to HLS, DASH, WebRTC, and other delivery protocols |
| `generative-video` | Produces or transforms video frames using AI models |
| `realtime-inference` | Runs AI model inference within the latency envelope of a live stream |
| `node-based-workflows` | Composes processing pipelines from connectable nodes |
| `creative-toolchain-integration` | Connects to professional AV tools via OSC, MIDI, Syphon, NDI |
| `content-provenance` | Cryptographically signs and tracks content origin and consent |
| `federated-identity` | Integrates with decentralised identity systems (AT Protocol, DID) |
| `social-video-distribution` | Routes video through open social protocol networks |
| `avatar-generation` | Creates or animates AI-driven digital human avatars |
| `lip-sync` | Synchronises avatar speech and facial animation to audio in real time |
| `interactive-presence` | Delivers conversational or responsive AI presence to end users |
