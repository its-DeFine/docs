## Cross-Tab Reference Seeker journey (primary)

| Step | What they're doing | What they need | Time |
|---|---|---|---|
| 1 | In origin tab with a question | Clear, fast reference lookup | 0s |
| 2 | Navigate to Resources | Sidebar link or search | <10s |
| 3 | Identify reference type | S1 routing: glossary / changelog / help / governance | <30s |
| 4 | Find specific entry | Scannable interface, search, filters | <60s |
| 5 | Return to origin tab | Answer in hand | <2 min total |

### Sub-journeys

**Glossary lookup (J1)**: Encounter term -> click glossary -> search/scroll -> read definition -> return. Target: <1 min.

**Changelog scan (J7)**: Wonder about breaking changes -> find component changelog -> scan recent entries -> assess risk. Target: <3 min.

**Source verification (J2)**: Read claim -> want to verify -> find primary source page -> confirm figure -> return with citation. Target: <5 min.

**Governance participation (J4)**: See LIP mentioned -> find LIP index -> read full text -> understand voting -> vote or decide. Target: 10-20 min.
