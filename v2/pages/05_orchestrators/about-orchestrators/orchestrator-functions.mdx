Orchestrators perform two main functions on the Livepeer network: transcoding and AI inference.
They do this by running a node that is connected to the network and performing the work that is assigned to it.

## Transcoding

Transcoding is the process of converting video from one format to another.
Orchestrators on the Livepeer network perform transcoding by running a node that is connected to the network and performing the work that is assigned to it.

## AI Inference

AI inference is the process of running a model to make a prediction. Orchestrators on the Livepeer network perform AI inference by running a node that is connected to the network and performing the work that is assigned to it.
