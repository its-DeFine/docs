# V2 Pages Structure

This folder contains all groupings relating to high-level TABS in the Livepeer Docs.

## Hierarchy

```
00_[NameOfTopLevelTab]
    00_[NameOfGroupSection]
        00_[NameOfPageContent.mdx]
    01_[NameOfNextSection]

01_[NameOfTopLevelTab]
    00_[NameOfGroupSection]
        00_[NameOfPageContent.mdx]
    01_[NameOfNextSection]
```

Top Level Groupings contain Sections.
Sections contain page content.

## The Top Level Groupings Are

- Home
- About
- Community
- Developers
- Gateways
- Orchestrators
- Delegators
