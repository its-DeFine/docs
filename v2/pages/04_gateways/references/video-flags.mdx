## Notes

- **Deprecated flags**: `-broadcaster` (use `-gateway` instead) 
- **Off-chain mode**: Use `-network offchain` to run without blockchain connectivity
- **Transcoding profiles**: Can be specified as presets (P720p30fps16x9) or JSON file path
- **HTTP ingest**: Automatically enabled when `-httpAddr` is localhost or when `-authWebhookUrl` is set
