Gateways can be configured to run both video transcoding and AI inference services in Livepeer.
This means

This section breaks down the Livepeer Gateway configuration space into two sets of capabilities:
- Transcoding Options (used by Mainnet Transcoding Network Gateways — often just called “Gateways”/“Broadcasters”)
- AI Gateway Flags & Options (used when running a Gateway in the Livepeer AI network context)

It will also share how to price these services.
