# Script Library

This repo uses a structured script taxonomy: `<type>/<concern>/<niche>`. All scripts live under `tools/scripts/`.

**Types:** audits (read-only scan/report), generators (produce artefacts), validators (pass/fail checks), remediators (bulk fix/repair), dispatch (orchestrate other scripts), automations (automated pipelines)

**Concerns:** content, components, governance, ai

---
