Use the dropdown below to switch between different views!

      ),
    },
    {
      title: "Linux/Mac",

        <>

## Position Options

You can control where the dropdown appears using the `position` prop:

- `position="inline"` - Default, appears inline with content
- `position="top-right"` - Fixed to top-right corner
- `position="top-left"` - Fixed to top-left corner

### Example with top-right position:

```jsx
<CustomViewDropdown
  position="top-right"
  views={[...]}
/>
```
